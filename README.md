Dependencias instaladas:
- Node.js -> npm init -y (pasta backend);
- Express
- Mongoose -> Para Banco de Dados
- CORS -> Para não ter problemas de requisição do mesmo dominio.
- -D nodemon -> Caso tenha alguma alteração ele irá atualizar sem que precise reinicar o servidor.

npm init -y
npm i express mongoose cors
npm i -D nodemon
